# ToDo Web App

## Opis
Prosta aplikacja webowa ToDo, która umożliwia dodawanie nowych zadań do listy oraz śledzenie postępu.

## Funkcje

- Dodawanie nowych zadań do listy
- Wyświetlanie listy zadań
- Podstawowe statystyki dotyczące ilości zadań
